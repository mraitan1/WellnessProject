# <center> DayByDay
### CD into client and server folders in separate terminal instances.
<ul>
    <li>Run 'npm run dev' in the client window</li>
    <li>Run 'npm start' in the server window</li>
</ul>

#### If you are getting error messages you might have to do these steps:

<ol>
    <li>Run 'npm install' in both client and server folders</li>
    <li>Might have to run 'npm audit fix' if there is an issue</li>
</ol>
